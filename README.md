# JJ2_NoteSpliter

JJ2のノートチェックで使う。

# Requirements

* Python 3.x
* MacOS

# INSTALL

```
$ brew tap ymmtr6/jj2_notespliter
$ brew install jj2_notespliter
```

# USAGE

```
$ jj2_notespliter [STUDENT_FILE] [NOTE_FILE]
```

詳しくは -h オプションで確認してください。

## STUDENT_FILE

学籍番号10桁、改行区切りのファイル

```
1210370999
1310370999


```
